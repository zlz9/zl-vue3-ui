# search 属性

| 属性     | 说明           | 默认值 |
| -------- | -------------- | ------ |
| listData | 搜索栏滚动数据 | -      |
| hotList  | 热门推荐       | -      |

# 事件

| 属性       | 说明                   | 参数     | 返回值 |
| ---------- | ---------------------- | -------- | ------ |
| search     | 搜索                   | 搜索信息 | void   |
| checkTitle | 点击标签和热门事件触发 | 标签     | void   |
